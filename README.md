# RPG Game Console V2.1

A console RPG game written in Python.

## Features

- Battle system
- Critical hit
- Dodge
- Inventory
- Equipment
- Loot system
- Healing items

## Requirements

- Python 3.14+
- Python-colorama>=0.4.6

# Installation

## Arch Linux

```bash
sudo pacman -S python-colorama
```

## Ubuntu/Debian

```bash
sudo apt install python3-colorama
```

## Windows/macOS

```bash
python -m pip install -r requirements.txt
```

## Run

```bash
python main.py
